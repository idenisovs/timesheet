# Activity Categorization

## Activity

Activity is the basic unit of work. Like you can fix a coffee machine once a year or attend a meeting with Wayne Enterprises representatives.

Let's imagine it like

```
2026-05-01
----------
09:00 - 10:30 - Meeting with Wayne Enterprises
10:30 - 11:00 - Coffee machine fix
11:00 - 12:00 - Breakfast
12:00 - 13:00 - Coffee machine fix
```
